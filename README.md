# 💼 Simran | Full Stack Developer

A modern and fully responsive developer portfolio built with **Next.js**, **Tailwind CSS**, and **GSAP**. This portfolio showcases my skills, projects, experience, achievements, and contact information with a clean UI and smooth animations.

## 🚀 Live Demo

🔗 https://portfolio-theta-bice-28.vercel.app/

---

## ✨ Features

- 🎨 Modern and responsive UI
- ⚡ Built with Next.js App Router
- 💨 Styled using Tailwind CSS
- 🎬 Smooth GSAP animations
- 📱 Fully responsive across all devices
- 📂 Featured and additional project showcase
- 💼 Professional experience timeline
- 🏆 Achievements & Certifications
- 📄 Resume download
- 📬 Contact section with social links

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/portfolio.git
```

Navigate to the project

```bash
cd portfolio
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open

```
http://localhost:3000
```
